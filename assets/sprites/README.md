Optional sprite override directory.

If present, Renderer3D will load these PNGs at runtime:
- grass.png
- water.png
- wall.png
- forest.png
- player_north.png
- player_east.png
- player_south.png
- player_west.png
- npc.png
- mob.png
- mob_dead.png

If a file is missing, a procedural placeholder sprite is generated in code.
